# SNSフォロワー数の週次更新

毎週月曜日に以下の手順でSNSデータを更新します。

## 対象タレント（9名）
1. 谷口彩菜
2. 吉富千桜
3. 中塚智
4. 五十嵐諒
5. 吉川慶
6. 小久保宏紀
7. 寺崎ひな
8. 田中希和
9. 島田和奏

## 更新手順

1. ブラウザで各タレントのTwitter/X、Instagram、TikTokを確認
2. フォロワー数をメモ
3. スプレッドシート `SNS_History` に新しい行を追加
   - A列: 日付 (例: 2025/12/23)
   - B列: タレント名
   - C列: Instagram
   - D列: TikTok
   - E列: X (Twitter)
4. 保存後、アプリは自動で最新データを反映

## ショートカット
「SNS更新」と依頼すれば、私が自動でデータ収集・更新を実行します。
